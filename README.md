## Noble Pitch Deck Site

This is a **static cinematic pitch deck** with a fixed “stage” and a defined number of scrollable screens.

### Run it

- **Fastest**: open `index.html` in your browser.
- **Recommended**: serve the folder with any static server so `fetch()` works reliably.

### Replace the pitch video

- Put your video at `assets/pitch.mp4`
- Optional poster: replace `assets/pitch-poster.svg`
